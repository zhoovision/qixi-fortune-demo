# 七夕互动抽签 Demo

一个适合手机竖屏体验的直播间礼物交互原型：

1. 在直播间礼物面板中选中「心有灵犀签」，签筒 icon 会在礼物格内轻轻摇晃。
2. 点击“送出”支付 9 钻，礼物面板直接收起，不播放额外的签筒动画页。
3. 随机祝福签以小卡片形式浮现在直播画面下方，完整展示 2 秒后渐隐消失。
4. 渐隐前点击签文卡片会重新打开礼物面板，再次送出后继续体验。

签文围绕用户与主播之间的直播关系随机生成，包括被记住、双向奔赴、日久陪伴、弹幕默契、并肩成长、例外偏爱和下次再见等类型。

签文卡片使用喜鹊、花枝与粉色吊绳组成的写实新中式福签，短签文以宋体与楷体竖排在下半段纸面留白中。

> iPhone 的动作传感器需要 HTTPS 环境和用户主动授权。GitHub Pages 发布后可直接使用；不支持动作传感器的浏览器会自动降级为轻触签筒触发。

## 本地运行

项目是纯静态网页，不需要安装依赖：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 发布

仓库已兼容 GitHub Pages，可从仓库的 `Settings → Pages` 选择 `Deploy from a branch`，并将分支设为 `main / root`。
